# Security Policy

## Supported Versions
| Version | Supported |
| ------ | --------- |
| main   | ✅        |

## Reporting a Vulnerability

If you find a security issue in this project, please contact us at: sicurezza@legalit.it

Please include:
- a description of the issue
- steps to reproduce
- any relevant logs or screenshots

We aim to respond within 3 working days and will keep you updated on the status of your report.
