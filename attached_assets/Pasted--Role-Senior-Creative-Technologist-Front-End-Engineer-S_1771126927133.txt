### Component API
```ts
type TopographicBackgroundProps = {
  interactive?: boolean; // default true
};
When interactive === false:

Do not attach the mousemove listener.

Keep u_mouse fixed at (0, 0) in [-1..1] space (or center), and do not apply cursor ripple.

Container styling
The React root element must be:

className="fixed inset-0 pointer-events-none z-0"

aria-hidden="true"

Renderer
alpha: true, antialias: true, powerPreference: 'high-performance'

setPixelRatio(Math.min(devicePixelRatio, 2))

Uniforms
u_time: float

u_mouse: vec2 in [-1..1]

u_resolution: vec2

u_color: vec3 from #2e6884

Shader behavior (GLSL)
Vertex: pass-through full-screen quad.

Fragment:

Build a pseudo height map using layered 2D noise.

Render anti-aliased contour lines (isolines) from the height map (minor + major lines).

Use u_color as base, but compute:

dark = base * 0.72

light = mix(base, vec3(1.0), 0.35)

A smooth tonal selector noise to blend dark → base → light so contour lines vary subtly but remain in the same blue family.

Mouse ripple:

Create a soft ripple around cursor using distance falloff and a sine term, and apply a very small domain warp.

Must feel “fluid” and gentle, not a harsh displacement.

Alpha:

Only lines should contribute alpha.

Keep overall opacity subtle; minor+major combined should stay in the ~0.2–0.35 feel (not opaque).

Cleanup (Strict Mode safe)
In the useEffect return:

remove resize listener

remove mousemove listener (only if attached)

cancelAnimationFrame

geometry.dispose()

material.dispose()

renderer.dispose()

remove renderer.domElement from container
