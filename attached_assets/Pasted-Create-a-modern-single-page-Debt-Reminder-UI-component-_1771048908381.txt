Create a modern, single-page "Debt Reminder" UI component.

Please implement the following HTML and CSS exactly, which sets up a specific staggered animation sequence (Entrance -> 0.3s Delay -> Color Sweep).

## 1. Project Files

**index.html**
Use a clean structure with a centered container.
Text content: "Saldo Totale: 62€ (12€ Latenti, 20€ Glovo, 30€ Ciock)"

**style.css**
Implement the following CSS logic to handle the keyframes:

```css
/* Main Layout */
body {
  margin: 0;
  height: 100vh;
  display: flex;
  justify-content: center;
  align-items: center;
  background-color: #f4f4f9; /* Light grey for contrast */
  font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif;
}

/* Container for the text */
.reminder-box {
  position: relative;
  overflow: hidden; /* Ensures the sweep bar doesn't show outside the text area */
  padding: 5px 10px;
}

/* The Target Element */
.debt-span {
  display: inline-block; /* Required for transform */
  font-size: 24px;
  font-weight: bold;
  color: #333;
  position: relative; /* Context for pseudo-element */
  
  /* Initial State */
  opacity: 0; 
  transform: translateY(-100%);
  
  /* Animation 1: Entrance */
  /* Duration: 0.8s | Timing: ease-out | Fill: forwards */
  animation: slideDownFade 0.8s ease-out forwards;
}

/* The Color Sweep Bar (Pseudo-element) */
.debt-span::after {
  content: "";
  position: absolute;
  top: 0;
  left: -100%; /* Start off-screen left */
  width: 100%;
  height: 100%;
  background-color: #083b6e; /* User specified color */
  opacity: 0.4; /* Slight transparency to see text under it, or remove for solid block */
  
  /* Animation 2: Sweep */
  /* Duration: 1.2s | Timing: ease-in-out */
  /* Delay calculation: 0.8s (entrance) + 0.3s (pause) = 1.1s total delay */
  animation: colorSweep 1.2s ease-in-out 1.1s forwards;
}

/* Keyframes Definitions */
@keyframes slideDownFade {
  0% {
    opacity: 0;
    transform: translateY(-100%);
  }
  100% {
    opacity: 1;
    transform: translateY(0);
  }
}

@keyframes colorSweep {
  0% {
    left: -100%;
  }
  100% {
    left: 100%;
  }
}
